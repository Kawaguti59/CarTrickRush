# Car Trick Rush

## このプロジェクトについて

**Car Trick Rush** は、車の空中回転（トリック）とスコアを軸にしたラン系の Unity ゲームです。WebGL 版は [Unity Room](https://unityroom.com/games/car-trick-rush) で公開しています（リポジトリは開発を継続）。

### 制作の目的

短期間で、一人でも一定以上のクオリティに届くゲームを仕上げることを目指した**個人開発のポートフォリオ**です。転職活動での活用も視野に入れつつ、実務で培った UI・ゲーム実装の経験を土台に、設計から実装までの幅を広げることも意図しています。

### 位置づけ

現時点のスキルを整理し、**設計力・実装力・開発効率**を確認するためのアウトプットです。今後のゲーム開発で必要になる「設計から実装まで一貫して考える力」の強化も目的としています。

**ひとことでいうと:** 設計から品質管理まで含めた、短期集中型の個人ゲーム開発プロジェクト。

### 開発で重視したこと

- **全体設計** — フォルダ構成・責務分離・依存関係を意識し、拡張性と保守性を考慮  
- **コーディング規約** — 命名・コメント・構造を事前に定め、プロジェクト内で統一  
- **短期間での品質** — 期間と優先順位を先に決め、実装の取捨選択を明確化  
- **プロセスの可視化** — Notion で簡易仕様・タスク管理し、フローを整理  
- **AI の活用** — 設計・実装・検証の補助として利用し、効率化を試行

## ゲーム概要
プレイヤーは自動で前進する車を操作し、空中で姿勢を変えてトリックを決め、コースをゴールまで走り切ります。  
制限時間やスコア、ギミック（ジャンプ台など）を通じて「走る・回る・稼ぐ」体験を目指しています。  

### 実装済み（例）
- 車両の基本挙動・自動前進  
- 入力による空中回転（トリック）とスコア  
- タイトル／ゲーム／ポーズ／リザルトのシーン遷移  
- BGM・効果音  

### 今後の予定（例）
- コース・ギミックの拡張  
- UI・難易度・チュートリアルの整理  
- WebGL 向け表示まわりの調整

## 起動方法（ブラウザでプレイ）

1. [Car Trick Rush（Unity Room）](https://unityroom.com/games/car-trick-rush) を開き、ページ内の手順に従ってプレイしてください。  
2. 対応ブラウザ（Chrome / Edge / Firefox など、WebGL が動作する環境）を推奨します。  

---

## エディタを用いる場合

1. 本リポジトリを Clone または Download ZIP してください。  
2. **Unity 6**（本プロジェクトはエディタ **6000.3 系** で作成）をインストールしてください。  
3. Unity Hub で本フォルダをプロジェクトとして開いてください。  
4. `BootScene`（または開発用の `BootSceneDebug`）から再生し、タイトル経由でゲームを開始できます。  

---

## 動作環境

- **Web 版（Unity Room）**: WebGL 対応のデスクトップブラウザ（モバイルは未検証の場合があります）  
- **エディタ開発**: Windows 10 / 11 推奨、Unity 6（6000.3 系）

## 操作方法

### キーボード

| 操作 | 割り当て |
|------|-----------|
| **車体を左に傾ける / 左回転** | **A** または **←** |
| **車体を右に傾ける / 右回転** | **D** または **→** |
| **車体を前に傾ける** | **W** または **↑** |
| **車体を後ろに傾ける** | **S** または **↓** |
| **ポーズ** | **Esc** または **Backspace**（ポーズ中にもう一度で再開する場合があります） |

### ゲームパッド

| 操作 | 割り当て |
|------|-----------|
| **車体を左に傾ける / 左回転** | **左スティック** 左 ／ **十字キー** 左 |
| **車体を右に傾ける / 右回転** | **右スティック** 右 ／ **十字キー** 右 |
| **車体を前に傾ける** | **左スティック** 上 ／ **十字キー** 上 |
| **車体を後ろに傾ける** | **左スティック** 下 ／ **十字キー** 下 |
| **ポーズ** | **Start** ボタン |
